# Dataset

## Source

Kaggle

Sentiment Analysis for Mental Health

https://www.kaggle.com/datasets/suchintikasarkar/sentiment-analysis-for-mental-health

## Description

The dataset contains over 53,000 labelled mental health-related text statements across seven sentiment categories.

Categories:

- Anxiety
- Bipolar
- Depression
- Normal
- Personality Disorder
- Stress
- Suicidal

The raw dataset is not redistributed in this repository.

Please download the dataset directly from Kaggle and place it under:

```
data/raw/raw_data.csv
```
